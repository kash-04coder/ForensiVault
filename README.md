# ForensiVault

This project aims to develop a secure digital evidence repository designed for cybercrime investigations. The system ensures proper handling, storage, and verification of digital evidence while maintaining a legally compliant chain of custody. 
Features: 
• Evidence Registration and Secure Storage 
• SHA-256 Hash Verification 
• Chain of Custody Tracking with Audit Logs 
• Metadata Extraction and Search Engine 
• Role-Based Access Control and Authentication
